# <a name="defender-for-iot-engine-alerts"></a>Оповещения защитника для механизма IoT

В этой статье описываются предупреждения, которые могут быть созданы из защитника для ядер IoT. Оповещения отображаются в окне оповещения, где можно управлять событием предупреждения.
