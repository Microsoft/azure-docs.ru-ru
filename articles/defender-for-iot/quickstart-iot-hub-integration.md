# <a name="get-started-with-built-in-iot-hub-integration"></a>Приступая к работе со встроенной интеграцией центра Интернета вещей

Этот параметр позволяет использовать службу без использования защитника для агентов безопасности IoT.

## <a name="enable-built-in-iot-hub-integration"></a>Включение встроенной интеграции центра Интернета вещей

Чтобы включить наблюдение за управлением удостоверениями устройств, устройствами в облаке и шаблонами связи "облако — устройство", выполните следующие действия, чтобы запустить службу.

1. Откройте **центр Интернета вещей**.
1. Откройте меню **Security Overview (Обзор безопасности** ).
1. Щелкните **защитить решение IOT** и заполните форму адаптации.

Поздравляем! Вы завершили включение защитника для службы IoT в центре Интернета вещей.
