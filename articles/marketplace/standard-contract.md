# <a name="standard-contract-for-microsoft-commercial-marketplace"></a>Стандартный контракт для коммерческой платформы Майкрософт

Корпорация Майкрософт предлагает стандартный контракт для коммерческой платформы Майкрософт. Это помогает упростить процесс приобретения клиентов, уменьшить юридические сложности для поставщиков программного обеспечения и упростить транзакции на платформе. Вместо того чтобы создавать собственные условия, как издатель на коммерческой платформе вы можете предложить свое программное обеспечение на основе [Стандартного контракта](https://go.microsoft.com/fwlink/?linkid=2041178), который клиентам надо просмотреть и принять лишь однажды.

Условия предложения определяются при создании предложения в центре партнеров. Вы можете решить использовать Стандартный контракт для коммерческой платформы Майкрософт вместо предоставления собственных условий.

>[!Note]
>После публикации предложения с помощью стандартного контракта для коммерческого рынка Майкрософт вы не сможете использовать собственные условия. Или вы предлагаете свое решение по Стандартному контракту, *или* на собственных условиях. Пользовательские условия определяются на уровне предложения и применяются ко всем планам. напишите свои пользовательские условия на странице **свойств** вашего предложения в центре партнеров. Если вы хотите изменить условия стандартного контракта, это можно сделать с помощью стандартных поправок контракта.

## <a name="standard-contract-amendments"></a>Поправки в Стандартный контракт

Поправки в Стандартный контракт позволяют выбрать, для простоты, Стандартный контракт, а затем модифицировать его для своего продукта или бизнеса. Клиентам необходимо будет изучить только поправки к стандартному контракту, если они уже изучили и приняли Стандартный контракт Майкрософт.

Для издателей на коммерческой платформе доступно два вида поправок:

* Универсальные поправки: применяются к стандартному контракту для всех клиентов. Универсальные поправки демонстрируются каждому клиенту в процессе покупки предложения. Чтобы использовать ваше предложение, клиенты должны принять условия Стандартного контракта и поправки.

* Индивидуальные поправки: это особые поправки к Стандартному контракту, ориентированные на конкретных клиентов по их идентификаторам клиентов Azure. Издатели могут выбирать, к каким клиентам будут применяться такие поправки. В процессе покупки условия с индивидуальными поправками будут предлагаться только клиентам с соответствующими идентификаторами.  Чтобы использовать ваше предложение, клиенты должны принять условия стандартного контракта и поправки (-ок).

>[!Note]
>Эти два типа поправок накладываются друг на друга. Клиенты, для которых предназначены индивидуальные поправки, в процессе покупки также сталкиваются с универсальными поправками. Длина ограничена 4000 символами, включая пробелы.

Вы можете использовать Стандартный контракт коммерческой платформы Майкрософт для следующих типов предложений:  приложения Azure (шаблоны решений и управляемые приложения), виртуальные машины и SaaS.

Во время ознакомления с предложениями в Azure Marketplace или AppSource клиенты будут видеть условия, связанные с предложением, как Стандартный контракт для коммерческой платформы Майкрософт, с любыми универсальными поправками.

## <a name="api"></a>API

Клиенты могут использовать Get-AzureRmMarketplaceTerms, чтобы получить условия предложения и принять его. Стандартный контракт и связанные с ним поправки будут возвращены в выходных данных командлета.
