# <a name="how-to-resell-your-offer-through-csp"></a>Как перепродать предложение через CSP

## <a name="resell-through-csp"></a>Перепродажа через CSP

На странице " **перепродажа по CSP** " (выберите в меню слева в центре партнеров), чтобы получить доступ к вашему предложению, сделав его доступным для партнеров в программе [поставщика облачных решений (CSP)](https://azure.microsoft.com/offers/ms-azr-0145p/) . Это позволяет торговым посредникам продавать ваше предложение своим клиентам и создавать пакетные решения. Все планы с использованием собственной лицензии (BYOL) включаются в эту программу автоматически. Вы также можете включить в нее другие планы.

Выберите партнеров, которые должны иметь возможность перепродажи вашего предложения:

- Любой партнер по программе CSP.
- Выбранные мной партнеры по программе CSP.
- Ни один из партнеров по программе CSP.

Выберите **Сохранить черновик** , прежде чем перейти к следующей вкладке в меню слева щелкните **тестовый накопитель**.
