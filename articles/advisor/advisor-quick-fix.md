# <a name="quick-fix-remediation-for-advisor"></a>Быстрое исправление исправлений для помощника
**Быстрое исправление** обеспечивает более быстрый и простой способ исправления для нескольких ресурсов. Она предоставляет возможность массового исправления ресурсов и помогает быстрее оптимизировать подписки с помощью исправления в масштабе ваших ресурсов.
Эта функция доступна только для определенных рекомендаций с помощью портал Azure.


## <a name="steps-to-use-quick-fix"></a>Инструкции по использованию быстрого исправления

1. В списке рекомендаций, имеющих метку **быстрого исправления** , щелкните рекомендацию.

   :::image type="content" source="./media/quick-fix-1.png" alt-text="{Снимок экрана помощника по Azure показывает метки быстрого исправления в рекомендациях.}":::
   
   *Цены на изображении предназначены только для примера*

2. На странице сведения об рекомендации вы увидите список ресурсов, для которых у вас есть эта рекомендация. Выберите все ресурсы, которые необходимо исправить для рекомендации.

   :::image type="content" source="./media/quick-fix-2.png" alt-text="Снимок экрана: окно затронутых ресурсов с элементами списка и кнопка быстрого исправления.":::
   
   *Цены на изображении предназначены только для примера*

3. Выбрав ресурсы, нажмите кнопку **быстрого устранения неполадок** , чтобы выполнить групповое исправление.

   > [!NOTE]
   > Некоторые из перечисленных ресурсов могут быть отключены, так как у вас нет соответствующих разрешений для их изменения.
   
   > [!NOTE]
   > Если существуют другие последствия, помимо преимуществ, упомянутых в помощнике, вы будете уведомлены об этом, чтобы помочь вам принять решения по устранению проблемы.
   
4. Вы получите уведомление о завершении исправления. Если в представлении списка ресурсов есть неисправленные ресурсы и ресурсы в выбранном режиме, вы увидите ошибку.
