# <a name="azure-policy-built-in-definitions-for-azure-data-box"></a>Встроенные определения политики Azure для Azure Data Box

Эта страница является индексом встроенных определений политик [Azure](../governance/policy/overview.md) для Azure Data Box. Дополнительные встроенные компоненты Политики Azure для других служб см. в статье [Встроенные определения Политики Azure](../governance/policy/samples/built-in-policies.md).
