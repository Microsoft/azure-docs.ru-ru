# <a name="automatically-identify-the-spoken-language-with-language-identification-model"></a>Автоматическое определение языка для речи с использованием модели идентификации языка

Индексатор видео поддерживает автоматическую идентификацию языка (крышки), которая представляет собой процесс автоматического определения содержания речевого языка с аудио и отправки файла мультимедиа для расшифрованной на основном определенном языке. 

В настоящее время крышка поддерживает следующие языки: Английский, испанский, французский, немецкий, итальянский, мандаринский диалект китайский, японский, русский и португальский (Бразилия). 

Обязательно ознакомьтесь с разделом [рекомендации и ограничения](#guidelines-and-limitations) ниже.

## <a name="choosing-auto-language-identification-on-indexing"></a>Выбор автоматического определения языка при индексировании

При индексировании видео или его [повторном индексировании](https://api-portal.videoindexer.ai/docs/services/operations/operations/Re-Index-Video?) с помощью API-интерфейса выберите `auto detect` параметр в `sourceLanguage` параметре.

При использовании портала перейдите на видео своей **учетной записи** на домашней странице [индексатора видео](https://www.videoindexer.ai/) и наведите указатель мыши на имя видео, для которого необходимо выполнить повторное индексирование. В правом нижнем углу нажмите кнопку Повторное индексирование. В диалоговом окне **Повторное индексирование видео** в раскрывающемся списке **язык источника видео** выберите *автоопределение* .

![Автоматическое обнаружение](./media/language-identification-model/auto-detect.png)

## <a name="model-output"></a>Выходные данные модели

Индексатор видео расшифровывает видео в соответствии с наиболее вероятным языком, если он имеет уверенность `> 0.6` . Если язык не может быть идентифицирован с уверенностью, предполагается, что речь составляет английский язык. 

Модель с главным языком доступна в формате JSON аналитики в качестве `sourceLanguage` атрибута (в разделе root/videos/Insights). В атрибуте также доступна соответствующая оценка достоверности `sourceLanguageConfidence` .

```json
"insights": {
        "version": "1.0.0.0",
        "duration": "0:05:30.902",
        "sourceLanguage": "fr-FR",
        "language": "fr-FR",
        "transcript": [...],
        . . .
        "sourceLanguageConfidence": 0.8563
      },
```

## <a name="guidelines-and-limitations"></a>Рекомендации и ограничения

* Автоматическая идентификация языка (крышка) поддерживает следующие языки: 

    Английский, испанский, французский, немецкий, итальянский, мандаринский диалект Чинес, японский, русский и португальский (Бразилия).
* Хотя индексатор видео поддерживает арабский (современные стандартные и Левантине), хинди и корейский, эти языки не поддерживаются в крышки.
* Если звук содержит языки, отличные от приведенного выше, результат будет непредвиденным.
* Если индексатор видео не может опознать язык с высокой достоверностью ( `>0.6` ), то для базового языка используется английский язык.
* Отсутствует текущая поддержка файла со звуком на смешанных языках. Если звук содержит смешанные языки, результат будет непредвиденным. 
* Низкое качество звука может повлиять на результаты модели.
* Для модели требуется по крайней мере одна минута речи в аудио.
* Модель разработана для распознавания неограниченного общения речевого ввода (не голосовых команд, подписания и т. д.).
