# <a name="examine-the-video-indexer-output"></a>Изучите выходные данные индексатора видео

При индексировании видео индексатор видео создает содержимое JSON, содержащее сведения об указанном видео Insights. Аналитические сведения включают в себя: записи, Окрс, лица, темы, блоки и т. д. Каждый тип Insights содержит экземпляры диапазонов времени, которые показывают, когда аналитика отображается в видео. 

Вы можете визуально просмотреть обобщенные аналитические сведения о видео, нажав кнопку **воспроизвести** в видео на веб-сайте [индексатора видео](https://www.videoindexer.ai/) . 

Вы также можете использовать API, вызвав API **получения индекса видео** , а состояние ответа — ОК. в качестве содержимого ответа вы получаете подробные выходные данные JSON.

![Аналитика](./media/video-indexer-output-json/video-indexer-summarized-insights.png)

В этой статье рассматриваются выходные данные индексатора видео (содержимое JSON). <br/>Сведения о доступных функциях и аналитических данных см. в статье [индексаторы видео](video-indexer-overview.md#video-insights).

> [!NOTE]
> Срок действия всех маркеров доступа в Индексаторе видео составляет один час.

## <a name="get-the-insights"></a>Получение ценной информации

### <a name="insightsoutput-produced-in-the-websiteportal"></a>Аналитика и выходные данные, созданные на веб-сайте или портале

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
1. Найдите в видеоролике выходные данные, которые необходимо изучить.
1. Нажмите кнопку **Воспроизвести**.
1. Выберите вкладку **аналитика** (сводка) или вкладку **временная шкала** (позволяет отфильтровать релевантную аналитику).
1. Загрузите артефакты и их содержимое.

Дополнительные сведения см. в руководстве по [просмотру и анализу аналитических сведений в Индексаторе видео](video-indexer-view-edit.md).

## <a name="insightsoutput-produced-by-api"></a>Аналитические сведения и вывод, созданные API

1. Чтобы получить JSON-файл, вызовите [API получения индекса видео](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Get-Video-Index?)
1. Если вы также заинтересованы в конкретных артефактах, вызовите функцию [получения видео API скачивания URL-адреса](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Get-Video-Artifact-Download-Url?)

    В вызове API укажите требуемый тип артефакта (OCR, Face, ключевые кадры и т. д.).

### <a name="insights"></a>insights

Каждая аналитика (например, строки транскрипции, лица, торговые марки и т. д.) содержит список уникальных элементов (например, face1, face2, face3), а каждый элемент имеет свои собственные метаданные и список его экземпляров (которые являются диапазонами времени с дополнительными необязательными метаданными).

У лица может быть идентификатор, имя, эскиз, другие метаданные и список временных экземпляров (например: 00:00:05 – 00:00:10, 00:01:00-00:02:30 и 00:41:21 – 00:41:49). Каждый временный экземпляр может иметь дополнительные метаданные. Например, координаты прямоугольной рамки вокруг лица (20,230,60,60).

|Версия|Версия кода|
|---|---|
|sourceLanguage|Исходный язык видео (предполагается один основной язык). В строковом формате на основе [BCP-47](https://tools.ietf.org/html/bcp47).|
|язык|Язык аналитических сведений (перевод с исходного языка). В строковом формате на основе [BCP-47](https://tools.ietf.org/html/bcp47).|
|transcript|Анализ [транскрипции](#transcript) .|
|ocr|Аналитика [OCR](#ocr) .|
|keywords|[Ключевые слова](#keywords) Insights.|
|blocks|Может содержать один или несколько [блоков](#blocks).|
|лица или Аниматедчарактерс|« [Лица» или «аниматедчарактерс](#facesanimatedcharacters) Insights».|
|метки;|Аналитика [меток](#labels) .|
|shots|Информация о [снимках](#shots) .|
|brands|Информация о [торговых марках](#brands) .|
|audioEffects|[Аудиоеффектс](#audioeffects-public-preview) Insights.|
|sentiments|[Тональности](#sentiments) Insights.|
|visualContentModeration|[Висуалконтентмодератион](#visualcontentmoderation) Insights.|
|textualContentModeration|[Текстуалконтентмодератион](#textualcontentmoderation) Insights.|
|emotions| [Эмоции](#emotions) Insights.|
|topics|Полезные [](#topics) сведения.|
|динамики|Информация о [колонках](#speakers) .|

Пример.

  "keywords": ...,
  "faces": ...,
  "labels": ...,
  "shots": ...,
  "brands": ...,
  "audioEffects": ...,
  "sentiments": ...,
  "visualContentModeration": ...,
  "textualContentModeration": ...
}

#### <a name="brands"></a>brands

Названия организаций и торговые марки продуктов, обнаруженные при преобразовании речи в текст и (или) распознавании текста на видео. Сюда не входит распознавание торговых марок или логотипов на изображении.

|name|Описание|

|name|Название торговой марки.|

|referenceUrl | URL-адрес статьи о торговой марке в Википедии (если существует). Например, [https://en.wikipedia.org/wiki/Target_Corporation](https://en.wikipedia.org/wiki/Target_Corporation) .
|description|Описание торговой марки.|
|tags|Список предопределенных тегов, которые связаны с этой торговой маркой.|
|confidence|Значение достоверности средства обнаружения торговых марок в Индексаторе видео (0–1).|
|instances|Список диапазонов времени, в которых упоминается эта торговая марка. Каждый экземпляр содержит объект brandType, который указывает, появилась ли торговая марка при расшифровке или распознавании текста.|

#### <a name="visualcontentmoderation"></a>visualContentModeration

Блок visualContentModeration содержит диапазоны времени, в которых Индексатор видео, вероятно, обнаружил содержимое для взрослых. Если блок visualContentModeration пуст, содержимое для взрослых не обнаружено.

Видео, в которых обнаружено содержимое для взрослых или содержимое непристойного характера, могут быть доступны только для закрытого просмотра. Пользователи могут отправить запрос для проверки содержимого человеком. В таком случае атрибут IsAdult будет содержать результат этой проверки.
