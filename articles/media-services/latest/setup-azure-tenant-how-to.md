# <a name="how-to-find-your-azure-tenant-id"></a>Как найти идентификатор вашего клиента в Azure?

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Поиск идентификатора клиента на портале

1. Войдите на [портал Azure](https://portal.azure.com).
1. Выберите значок Azure Active Directory в разделе службы Azure. Откроется страница Active Directory.
1. Идентификатор клиента находится в разделе **Основные сведения**.
1. Скопируйте идентификатор клиента и вставьте его в текстовый документ по своему усмотрению для последующего использования.
