> [!NOTE]
> Учетные записи, созданные с помощью API 2020-05-01, не могут использовать устаревший API RESTv2.  Более старые версии API можно использовать для создания учетных записей, которые по-прежнему могут работать с устаревшим API RESTv2, если это необходимо.<br/><br/>
> Они имеют доступ к новым функциям, таким как управляемое удостоверение, шифрование неактивных данных с помощью управляемого клиентом ключа и предоставление доверенного доступа к службе из учетной записи служб мультимедиа в подключенные учетные записи хранения.
