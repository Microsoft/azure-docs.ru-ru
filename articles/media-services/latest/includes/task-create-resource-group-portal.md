## <a name="create-a-resource-group"></a>Создание группы ресурсов

1. На начальном экране портала Azure выберите **Создать ресурс**. Появится окно Marketplace.
1. Выберите элемент **Группы ресурсов**.  Отобразится список групп ресурсов.
1. Выберите **Добавить**. Появится область для создания группы ресурсов.
1. Выберите подписку, которая будет использоваться для этой группы ресурсов.
1. В поле **Группа ресурсов** введите имя группы ресурсов.
1. Выберите **регион** для группы ресурсов.
1. Выберите **Review + create** (Просмотреть и создать).
