# <a name="azure-spatial-anchors-overview"></a>Описание службы "Пространственные привязки Azure"

Вас приветствует служба "Пространственные привязки Azure". Служба "Пространственные привязки" Azure предоставляет разработчикам необходимые возможности для создания пространственных приложений смешанной реальности. Эти приложения могут работать на устройствах Майкрософт HoloLens, iOS с поддержкой ARKit и устройствах Android с поддержкой ARCore. Служба "Пространственные привязки" Azure позволяет разработчикам работать с платформами смешанной реальности, чтобы взаимодействовать с пространствами, точно задавать точки интереса и вызывать эти точки интереса с поддерживаемых устройств.
Эти точки интереса называются пространственными привязками.

![Кроссплатформенная поддержка](./media/cross-platform.png)

## <a name="examples"></a>Примеры

Некоторые варианты использования службы "Пространственные привязки" описаны ниже:

- [Многопользовательский режим](tutorials/tutorial-share-anchors-across-devices.md). Служба "Пространственные привязки Azure" облегчает пользователям, находящимся в одном и том же расположении, работу с многопользовательскими приложениями смешанной реальности. Например, два пользователя могут запустить игру в шахматы на основе смешанной реальности, разместив виртуальную шахматную доску на столе. Затем, наведя свое устройство на стол, пользователи смогут видеть и взаимодействовать с виртуальной шахматной доской.

- [Поиск пути](concepts/anchor-relationships-way-finding.md). Разработчики также могут соединять пространственные привязки, создавая связи между ними. Например, в приложении могут быть сцены с двумя или более точками интереса, с которыми пользователь должен взаимодействовать для выполнения задачи. Эти точки интереса могут создаваться в подключенном режиме. Позже, когда пользователь завершит задачу, состоящую из нескольких этапов, приложение может запросить ближайшие привязки, чтобы направить пользователя к следующему этапу задачи.

- [Сохранение виртуального содержимого в реальной жизни](how-tos/create-locate-anchors-unity.md#create-a-cloud-spatial-anchor). С помощью приложения пользователь может разместить виртуальный календарь на стене конференц-зала, который люди могут видеть с помощью приложения для телефона или устройства HoloLens. В промышленных условиях пользователь может получать контекстную информацию об оборудовании, направив на него камеру поддерживаемого устройства.

Служба "Пространственные привязки Azure" состоит из управляемой службы и клиентского пакета SDK для платформ поддерживаемых устройств. В следующих разделах показано, как начать создавать приложения с использованием службы "Пространственные привязки Azure".
