# <a name="what-is-the-autoregistration-feature-of-azure-dns-private-zones"></a>Что такое функция авторегистрации в Azure DNS частных зонах

Функция автоматической регистрации Azure DNS частные зоны требует от вас отменять управление записями DNS для виртуальных машин, развернутых в виртуальной сети. Когда вы [связываете виртуальную сеть](./private-dns-virtual-network-links.md) с частной зоной DNS и включаете автоматическую регистрацию для всех виртуальных машин, записи DNS для виртуальных машин, развернутых в виртуальной сети, автоматически создаются в частной зоне DNS. Кроме записей просмотра (записей), записи обратного просмотра (записи PTR) также создаются автоматически для виртуальных машин.
При добавлении в виртуальную сеть дополнительных виртуальных машин записи DNS для этих виртуальных машин также автоматически создаются в связанной частной зоне DNS.

При удалении виртуальной машины записи DNS для виртуальной машины автоматически удаляются из частной зоны DNS.

Можно включить автоматическую регистрацию, выбрав параметр "включить авторегистрацию" при создании связи виртуальной сети.

![Включить автоматическую регистрацию](./media/privatedns-concepts/enable-autoregistration.png)

## <a name="restrictions"></a>Ограничения

* Функция авторегистрации работает только для виртуальных машин. Для всех других ресурсов, таких как внутренние подсистемы балансировки нагрузки и т. д., можно создавать записи DNS вручную в частной зоне DNS, связанной с виртуальной сетью.
* Записи DNS создаются автоматически только для основной сетевой карты виртуальной машины. Если на виртуальных машинах имеется несколько сетевых адаптеров, можно вручную создать записи DNS для других сетевых интерфейсов.
* Записи DNS создаются автоматически, только если основной сетевой адаптер виртуальной машины использует DHCP. Если настроены статические IP-адреса (например, для использования [нескольких IP-адресов в Azure](../virtual-network/virtual-network-multiple-ip-addresses-portal.md#os-config)), при авторегистрации не создаются записи для этой виртуальной машины.
* Авторегистрация для IPv6 (записи AAAA) не поддерживается.
