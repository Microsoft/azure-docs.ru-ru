# <a name="azure-stack-edge-pro-technical-specifications"></a>Технические спецификации Azure Stack пограничных Pro

Аппаратные компоненты устройства Microsoft Azure Stack пограничной Pro соответствуют техническим спецификациям и нормативным стандартам, изложенным в этой статье. В технических спецификациях описываются источники питания, габаритные размеры, емкость и стандарты охраны окружающей среды.

## <a name="compute-memory-specifications"></a>Спецификации вычислительных компонентов и запоминающих устройств

Azure Stack пограничных устройств Pro имеет следующие спецификации для вычислений и памяти.

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| ЦП    | 2 X 10 ядер ЦП Intel Xeon серебро 4114 2.2 G                    |
| Память              | 128 ГБ ОЗУ (8X 16 ГБ RDIMM)                 |

## <a name="fpga-specifications"></a>Спецификации ППВМ

Поле с программируемым набором шлюзов (FPGA) включено на каждом устройстве Azure Stack пограничной Pro, которое обеспечивает сценарии Машинное обучение (машинного обучения).

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| ППВМ   | Intel Arria 10 <br> Доступные модели с поддержкой глубокой нейронной сети (DNN) те же, что и [экземпляры ППВМ с поддержкой облака](../machine-learning/how-to-deploy-fpga-web-service.md#fpga-support-in-azure).|

## <a name="power-supply-unit-specifications"></a>Спецификации источника питания

Устройство Azure Stack пограничной Pro имеет два блока питания 100-240 V (блока питания) с высокопроизводительными вентиляторами. Два таких источника питания обеспечивают конфигурацию с избыточным питанием. В случае сбоя источника питания устройство продолжает работать в обычном режиме на другом источнике питания до замены отказавшего модуля. В таблице ниже приведены технические спецификации модулей питания.

| Спецификация           | БП 750 ВТ                  |
|-------------------------|----------------------------|
| Максимальная выходная мощность    | 750 Вт                     |
| Частота               | 50/60 Гц                   |
| Выбор диапазона напряжения | Автоматический выбор напряжения: 100–240 В переменного тока |
| «Горячая» замена           | Да                        |

### <a name="azure-stack-edge-pro-power-cord-specifications-by-region"></a>Спецификации кабелей питания Pro Azure Stack для региона

Устройству Azure Stack пограничной Pro требуется кабель питания, который зависит от региона Azure.
Технические спецификации всех поддерживаемых кабелей питания см. [в разделе спецификации кабелей питания Pro Azure Stack](azure-stack-edge-technical-specifications-power-cords-regional.md).

## <a name="storage-specifications"></a>Спецификации хранилища

На устройствах Azure Stack пограничных Pro установлено 9 X 2,5 "твердотельных накопителей NVMe, каждый из которых имеет емкость 1,6 ТБ. Из этих твердотельных накопителей 1 — это диск операционной системы, а оставшиеся 8 — это диски данных. Общая полезная емкость устройства составляет примерно 12,5 ТБ. В таблице ниже приведены подробные сведения о емкости хранилища устройства.

|     Спецификация                          |     Значение             |
|--------------------------------------------|-----------------------|
|    Количество твердотельных накопителей (SSD)     |    8                  |
|    Емкость одного твердотельного накопителя (SSD)                     |    1,6 TБ             |
|    Общая емкость                          |    12,8 ТБ            |
|    Общая доступная емкость*                  |    ~ 12,5 ТБ            |

**Определенное пространство зарезервировано для внутреннего использования.*
