# <a name="azure-stack-edge-pro-with-fpga-2101-release-notes"></a>Azure Stack ребра Pro с заметками о выпуске FPGA 2101

В следующих заметках о выпуске указываются критические открытые проблемы и Устраненные проблемы, связанные с выпуском 2101 Azure Stack ребра Pro со встроенным программируемым массивом шлюзов (FPGA).

Заметки о выпуске постоянно обновляются. При обнаружении критических проблем, требующих обхода, они добавляются. Перед развертыванием Azure Stack пограничной устройства внимательно ознакомьтесь с информацией в заметках о выпуске.  

Этот выпуск соответствует версии программного обеспечения:

- **Azure Stack пограничных 2101 (1.6.1475.2528)** — KB 4599267

> [!NOTE]
> Обновление 2101 можно применить только к устройствам, на которых выполняются общедоступные версии программного обеспечения или более поздней версии.

## <a name="whats-new"></a>Новые возможности

Этот выпуск содержит исправление следующей ошибки.

- **Отправка проблемы** . в этом выпуске Исправлена проблема с отправкой, при которой перезагрузка из-за сбоя может привести к снижению скорости выполнения передачи. Эта проблема может возникнуть при отправке набора данных, который в основном состоит из файлов, больших относительно доступной пропускной способности, особенно без ограничений, при активном регулировании полосы пропускания. Это изменение обеспечивает достаточную возможность для завершения отправки перед перезапуском отправки для заданного файла.

Этот выпуск также содержит следующие обновления:

- Все накопительные обновления Windows и обновления .NET Framework, выпущенные до октября 2020 г.
- Версия встроенного по контроллера управления основной платой (BMC) обновляется с 3.32.32.32 до 3.36.36.36 во время установки фабрики для устранения несовместимости с более новыми единицами питания Dell.
- Этот выпуск поддерживает IoT Edge 1.0.9.3 на Azure Stack пограничных устройствах.
