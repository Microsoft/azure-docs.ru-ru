# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Руководство по началу разработки встраиваемого устройства Интернета вещей Azure

**Область применения**: [разработка встраиваемых устройств](about-iot-develop.md#embedded-device-development)

Это руководство по началу работы содержит набор кратких руководств по началу работы со встраиваемыми устройствами и Интернетом вещей Azure. 

Используя инструкции из этого краткого руководства, вы выполните следующие основные задачи:
* Установка набора встроенных средств разработки для программирования конкретного устройства на языке C.
* Создание образа, включающего компоненты и примеры ОСРВ Azure, и программирование устройства.
* Безопасное подключение устройства к Интернету вещей Azure.
* Просмотр данных телеметрии устройства и свойств, а также вызов методов "из облака на устройство".
