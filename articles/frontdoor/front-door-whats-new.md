# <a name="whats-new-in-azure-front-door"></a>Новые возможности в Azure Front Door

Служба Azure Front Door регулярно обновляется. Чтобы вы оставались в курсе последних разработок, в этой статье предоставлены такие сведения:

- Последние выпуски.
- Известные проблемы
- Исправления ошибок
- Нерекомендуемые функции.

## <a name="new-features"></a>новые функции;

|Компонент  |Описание  |Дата добавления  |
|---------|---------|---------|
| Общедоступная версия обработчика правил | Настройка обработки HTTP-запросов в пограничной среде. Дополнительные сведения см. в [обзоре обработчика правил](front-door-rules-engine.md). |Июнь 2020 г. |
| Обработчик правил (предварительная версия) | Настройка обработки HTTP-запросов в пограничной среде. Дополнительные сведения см. в [обзоре обработчика правил](front-door-rules-engine.md). |Апрель 2020 г. |
