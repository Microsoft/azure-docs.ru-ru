> [!NOTE]
> Начиная с сентября 22 2020, большинство контейнеров с условным просмотром размещаются в реестре контейнеров Майкрософт, и для их загрузки не требуется использовать команду DOCKER Login. Вам по-прежнему потребуется выполнить запрос в Интернете для запуска контейнера. Дополнительные сведения см. в разделе **запрос утверждения для выполнения контейнера** далее в этой статье.
