Одной из основных функций службы "Речь" является распознавание и преобразование человеческой речи (часто это называется преобразованием речи в текст). Из этого примера вы узнаете, как использовать пакет SDK для службы "Речь" в приложениях и продуктах для выполнения высококачественного преобразования речи в текст.

## <a name="react-sample-on-github"></a>Пример React на сайте GitHub

Перейдите к [примеру React](https://github.com/Azure-Samples/AzureSpeechReactSample) на сайте GitHub, чтобы узнать, как использовать пакет SDK службы "Речь" в среде JavaScript на основе браузера. В этом примере приведены примеры шаблонов разработки для обмена маркерами аутентификации и управления ими, а также записи звука с микрофона или из файла для преобразования речи в текст.

Кроме того, шаблоны разработки, используемые в [кратком руководстве по Node.js](https://docs.microsoft.com/azure/cognitive-services/speech-service/get-started-speech-to-text?tabs=script%2Cbrowser%2Cwindowsinstall&pivots=programming-language-nodejs), можно использовать в среде на основе браузера.
