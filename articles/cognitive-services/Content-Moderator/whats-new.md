# <a name="whats-new-in-content-moderator"></a>Новые возможности Content Moderator

Узнайте о новых возможностях службы. Эти элементы могут включать заметки о выпуске, видео, записи блогов и другие типы информации. Создайте закладку для этой страницы, чтобы получать последние сведения об обновлениях службы.

## <a name="august-2020"></a>Август 2020 г.

### <a name="new-video-moderation-ui"></a>Новый пользовательский интерфейс для модерации видео

* На страницу модерации видео в средстве проверки добавлены новые инструменты и функции, упрощающие назначение тегов и обработку видео или его частей. Подробные сведения см. в [руководстве по модерации видео](./video-moderation-human-review.md).

### <a name="new-dashboard"></a>Новая панель мониторинга

* Панель мониторинга в средстве проверки теперь отображает дополнительные сведения об открытых, ожидающих завершения и завершенных проверках. Обзор новых функций см. в [руководстве по средству проверки](./Review-Tool-User-Guide/human-in-the-loop.md).
