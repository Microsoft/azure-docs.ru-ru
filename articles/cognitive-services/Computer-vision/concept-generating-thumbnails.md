# <a name="generating-smart-cropped-thumbnails-with-computer-vision"></a>Использование компьютерного зрения для создания эскизов с примененной интеллектуальной обрезкой

Эскиз — это уменьшенное представление изображения. Эскизы используются для представления изображений и других данных как более экономичный в плане занимаемого места способ. Используя интеллектуальный алгоритм обрезки изображения и изменяя его размеры, API компьютерного зрения создает для заданного изображения понятные эскизы.

Алгоритм создания эскизов в службе "Компьютерное зрение" работает следующим образом:

1. Удаление ненужных элементов из изображения и указание _области интересов_ для &mdash; области изображения, в котором отображаются основные объекты.
1. Обрезает изображение, исходя из определенной _зоны внимания_.
1. Изменяет пропорции в соответствии с размерами требуемого эскиза.

## <a name="area-of-interest"></a>Зона внимания

Когда вы отправили изображение, API компьютерного зрения анализирует его, пытаясь определить *зону внимания*. Затем на основании выбранной зоны изображение обрезается. При обрезке всегда учитываются требуемые пропорции, если они указаны.

Кроме того, координаты прямоугольника, ограничивающего выбранную *зону внимания*, можно получить с помощью вызова API **areaOfInterest**. Получив эти сведения, вы можете как угодно изменять исходное изображения.
