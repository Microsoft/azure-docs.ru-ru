# <a name="extend-azure-with-solutions-from-partners"></a>Расширьте возможности Azure с помощью решений от партнеров

Партнерские организации предлагают решения, которые можно использовать в Azure для улучшения облачной инфраструктуры. Эти решения полностью интегрированы в Azure. Вы работаете с этими решениями практически так же, как вы работаете с решениями от Майкрософт. Для управления решением используется поставщик ресурсов, типы ресурсов и пакеты SDK.

Решения партнеров доступны в Marketplace.

| Партнерское решение | Описание |
| :--- | :--- |
| [Apache Kafka для Fluent в облаке](./apache-kafka-confluent-cloud/overview.md) | Полностью управляемая платформа потоковой передачи событий на базе Apache Kafka |
