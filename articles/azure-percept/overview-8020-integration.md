# <a name="azure-percept-dk-8020-integration-overview"></a>Обзор интеграции Azure Перцепт DK 80/20

Устройство Azure Перцепт DK и аудиоустройство разрабатывались для интеграции с [системой салазок 80/20](https://8020.net/).

## <a name="8020-features"></a>компоненты 80/20

Каждый аппаратный компонент строится с учетом и протрусионс в соответствии с типом объема 1010. Такая интеграция позволяет клиентам и сборщикам решений более легко расширять свои концепции в рабочих средах.
