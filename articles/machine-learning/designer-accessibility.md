# <a name="use-a-keyboard-to-use-azure-machine-learning-designer"></a>Использование клавиатуры для использования конструктора Машинное обучение Azure

Узнайте, как использовать клавиатуру и средство чтения с экрана для работы в конструкторе Машинного обучения Azure. Список сочетаний клавиш, которые работают везде на портале Azure, приведен в разделе [Сочетания клавиш на портале Azure](../azure-portal/azure-portal-keyboard-shortcuts.md).

Этот рабочий процесс тестировался с помощью [Narrator](https://support.microsoft.com/help/22798/windows-10-complete-guide-to-narrator) и [JAWS](https://www.freedomscientific.com/products/software/jaws/), но он должен подойти и для других стандартных средств чтения с экрана.

## <a name="navigate-the-pipeline-graph"></a>Перемещение по графу конвейера

Граф конвейера организован как вложенный список. Внешний список — это список модулей, который описывает все модули в графе конвейера. Внутренний список — это собой список подключений, в котором описаны все подключения определенного модуля.  

1. В списке модулей с помощью клавиши со стрелкой можно переключать модули.
1. Используйте вкладку, чтобы открыть список подключений для целевого модуля.
1. Используйте клавишу со стрелкой для переключения между портами подключения модуля.
1. Используйте клавишу G для перехода к целевому модулю.

## <a name="edit-the-pipeline-graph"></a>Изменение графа конвейера

### <a name="add-a-module-to-the-graph"></a>Добавление модуля в граф

1. Нажмите клавиши CTRL+F6, чтобы переключить фокус с холста на дерево модулей.
1. Найдите нужный модуль в дереве модулей, используя стандартный элемент управления иерархического представления.

### <a name="edit-a-module"></a>Изменение модуля

Чтобы подключить модуль к другому модулю, сделайте следующее.

1. Нажмите клавиши CTRL+SHIFT+H при выборе модуля в списке модулей, чтобы открыть вспомогательное приложение подключения.
1. Измените порты подключения для модуля.

Чтобы настроить свойства модуля, сделайте следующее.

1. Нажмите клавиши CTRL+SHIFT+E при выборе модуля, чтобы открыть его свойства.
1. Измените свойства модуля.

## <a name="navigation-shortcuts"></a>Сочетания клавиш для перемещения

| Клавиши | Описание |
|-|-|
| CTRL+F6 | Переключение фокуса между холстом и деревом модулей. |
| CTRL+F1   | Открыть информационную карточку при фокусировании на узле в дереве модулей. |
| CTRL+SHIFT+H | Открыть вспомогательное приложение подключения, если фокус находится на узле. |
| CTRL+SHIFT+E | Открыть свойства модуля, если фокус находится на узле. |
| CTRL+G | Переместить фокус на первый сбойный узел в случае сбоя выполнения конвейера. |

## <a name="action-shortcuts"></a>Ярлыки действий

Используйте следующие сочетания с клавишей доступа. Дополнительные сведения о клавишах доступа: https://en.wikipedia.org/wiki/Access_key.

| Клавиши | Действие |
|-|-|
| Клавиша доступа + R | Run (Запустить) |
| Клавиша доступа + P | Публикация |
| Клавиша доступа + C | Clone (Клонировать) |
| Клавиша доступа + D | Развертывание |
| Клавиша доступа + I | Создать или обновить конвейер вывода. |
| Клавиша доступа + B | Создать или обновить конвейер пакетного вывода. |
| Клавиша доступа + K | Открыть раскрывающийся список "Create inference pipeline" (Создание конвейера вывода). |
| Клавиша доступа + U | Открыть раскрывающийся список "Update inference pipeline" (Обновление конвейера вывода). |
| Клавиша доступа + M | Открыть раскрывающийся список дополнительных функций (…). |
