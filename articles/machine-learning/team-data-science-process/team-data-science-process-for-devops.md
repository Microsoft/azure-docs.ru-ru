# <a name="team-data-science-process-for-developer-operations"></a>Процесс обработки и анализа данных группы для операций разработки

В этой статье рассматриваются функции операций разработки (DevOps), которые относятся к реализации решения расширенной аналитики и Cognitive Services. В этих учебных материалах реализуется процесс обработки и анализа данных группы (TDSP), а также программное обеспечение и наборы средств с открытым исходным кодом, которые полезны для представления, выполнения и доставки решений для обработки и анализа данных. Они ссылаются на разделы, посвященные цепочке инструментов DevOps, характерной для проектов и решений для обработки и анализа данных и Application Insights.
