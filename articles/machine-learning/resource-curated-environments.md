# <a name="azure-machine-learning-curated-environments"></a>Машинное обучение Azure проверенные среды

В этой статье перечислены проверенные среды в Машинное обучение Azure. Проверенные среды предоставляются Машинное обучение Azure и доступны в рабочей области по умолчанию. Они поддерживаются кэшированными образами DOCKER, которые используют последнюю версию пакета SDK для Машинное обучение Azure, уменьшая затраты на подготовку к запуску и позволяя ускорить развертывание. Используйте эти среды для быстрого начала работы с различными платформами машинного обучения.

> [!NOTE]
> Этот список обновляется по состоянию на декабрь 2020. Используйте пакет SDK для Python, чтобы получить самый обновленный список сред и их зависимостей. Дополнительные сведения см. в [статье о средах](./how-to-use-environments.md#use-a-curated-environment).
