Проблемы, связанные со службой, плановое обслуживание, рекомендации по работоспособности, рекомендации по обеспечению безопасности, уведомления публикуются в разделе **Работоспособность служб** на портале Azure. После настройки оповещений для журнала действий для таких уведомлений можно будет выполнять своевременные действия. Дополнительные сведения см. в разделе [Создание оповещений о работоспособности служб с помощью портала Azure](../../service-health/alerts-activity-log-service-notifications-portal.md#create-service-health-alert-using-azure-portal).
