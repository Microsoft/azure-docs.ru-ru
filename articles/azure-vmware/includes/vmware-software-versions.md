Текущие версии программного обеспечения VMware, используемые в кластерах частных облачных решений Azure VMware:

| Программное обеспечение              |    Версия   |
| :---                  |     :---:    |
| ВКСА/vSphere/ESXi |    6,7 U3    | 
| ESXi                  |    6,7 U3    | 
| vSAN                  |    6,7 U3    |
| НСКС-T                 |      2.5     |


>[!NOTE]
>В настоящее время поддерживается только версия NSX-T.

Для любого нового кластера в частном облаке версия программного обеспечения соответствует тому, что выполняется в данный момент. Для любого нового частного облака в подписке устанавливается последняя версия программного стека. Дополнительные сведения см. в статье [требования к версии программного обеспечения VMware](https://docs.vmware.com/en/VMware-HCX/services/user-guide/GUID-54E5293B-8707-4D29-BFE8-EE63539CC49B.html).

Обновления пакетов для частного облака сохраняют программное обеспечение в одной версии пакета последних версий программного обеспечения из VMware. Версии программного обеспечения частного облака могут отличаться от самых последних версий отдельных программных компонентов (ESXi, НСКС-T, vCenter, vSAN). Общие политики обновления и процессы для программного обеспечения платформы решений VMware для Azure, описанные в разделе [обновления и обновление частного облака](../concepts-upgrades.md), можно найти в этой статье.
