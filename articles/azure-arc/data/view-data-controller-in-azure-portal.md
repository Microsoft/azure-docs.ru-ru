# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Просмотр ресурса контроллера данных Arc Azure в портал Azure

После того как первые [метрики или журналы](upload-metrics-and-logs-to-azure-monitor.md) будут отправлены в Azure или [переданы сведения об использовании](view-billing-data-in-azure.md), вы сможете увидеть, что ресурс контроллера данных ARC в Azure создан в портал Azure.

Откройте портал Azure с помощью специального URL-адреса:  [https://portal.azure.com](https://portal.azure.com) .

Выполните поиск контроллера данных по имени в строке поиска и щелкните его.
