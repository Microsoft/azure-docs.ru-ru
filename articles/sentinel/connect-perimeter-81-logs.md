# <a name="connect-your-perimeter-81-activity-logs-to-azure-sentinel"></a>Подключение журналов действий периметра 81 к Azure Sentinel

> [!IMPORTANT]
> Соединитель данных периметра 81 в Azure Sentinel сейчас находится в общедоступной предварительной версии.
> Эта функция предоставляется без соглашения об уровне обслуживания и не рекомендуется для рабочих нагрузок. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

В этой статье объясняется, как подключить устройство [журналов действий периметра 81](https://www.perimeter81.com/) к Azure Sentinel. Соединитель журналов действий периметра 81 позволяет легко перенести данные периметра 81 в метку Azure, чтобы их можно было просматривать в книгах, использовать для создания пользовательских оповещений и внедрять их для улучшения расследования.

> [!NOTE]
> Данные будут храниться в географическом расположении рабочей области, на которой вы используете метку Azure.

## <a name="prerequisites"></a>Предварительные условия

- У вас должны быть разрешения на чтение и запись в рабочей области Sentinel Azure.

- Необходимо иметь разрешения на чтение общих ключей для рабочей области.

## <a name="configure-and-connect-perimeter-81-activity-logs"></a>Настройка и подключение журналов действий для периметра 81

Журналы действий периметра 81 могут интегрировать и экспортировать журналы непосредственно в Azure Sentinel.

1. На портале Sentinel Azure щелкните **соединители данных** в меню навигации.

1. Выберите в коллекции **журналы действий периметра 81** и нажмите кнопку **открыть страницу соединителя** .

1. На странице соединитель журналов действий периметра 81 скопируйте **идентификатор рабочей области** и **первичный ключ** и вставьте их в периметре 81, [как описано здесь](https://support.perimeter81.com/hc/en-us/articles/360012680780).

1. После выполнения инструкций вы увидите типы подключенных данных на странице соединителя Azure Sentinel.

## <a name="find-your-data"></a>Поиск данных

После установки успешного подключения данные отображаются в **журналах** в разделе **CustomLogs**  -  **Perimeter81_CL**.

После того, как журналы начнут отображаться, может пройти до 20 минут.
