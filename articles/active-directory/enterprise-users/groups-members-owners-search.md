# <a name="search-groups-and-members-preview-in-azure-active-directory"></a>Поиск групп и участников (Предварительная версия) в Azure Active Directory

В этой статье содержатся сведения о поиске членов и владельцев группы, а также об использовании фильтров поиска в рамках предварительной версии улучшения групп на портале Azure Active Directory (Azure AD). Существует множество улучшений в работе групп, которые помогут вам быстро и легко управлять группами, включая членов и владельцев. См. подробные сведения о [дополнительных условиях использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

К изменениям в этой предварительной версии относятся:

- Новые возможности поиска групп, такие как поиск подстрок в именах групп
- Новые параметры фильтрации и сортировки в списках членов и владельцев
- Новые возможности поиска для списков членов и владельцев
- Более точные счетчики групп для больших групп

## <a name="enabling-and-managing-the-preview"></a>Включение и управление предварительной версией

Мы упростили присоединение к предварительной версии:

  1. Войдите на [портал Azure AD](https://portal.azure.com)и выберите **группы**.
  2. На странице группы — все группы выберите баннер в верхней части страницы, чтобы присоединиться к предварительной версии.

Вы также можете просмотреть последние функции и улучшения, выбрав **Предварительные сведения** на странице **все группы** . После приподключения к предварительной версии можно увидеть тег Preview на всех страницах групп, которые имеют улучшения и являются частью предварительной версии. В рамках этой предварительной версии была обновлена не каждая страница групп.

Если у вас возникли какие-либо проблемы, можно вернуться к предыдущей версии, выбрав баннер в верхней части страницы **все группы** . Мы ценим ваши отзывы, чтобы мы могли улучшить наш опыт.

## <a name="group-search-and-sorting"></a>Поиск и сортировка групп

Поиск в списке групп был усовершенствован, так что при вводе строки поиска Поиск автоматически выполняет `startswith` Поиск и подстроку в списке имен групп. Поиск подстроки выполняется только для целых слов и не содержит специальных символов. При поиске подстроки учитывается регистр.

![Новый поиск по подстрокам на странице "все группы"](./media/groups-members-owners-search/groups-search-preview.png)

Например, Поиск «Policy» теперь будет возвращать "политики MDM — Западная" и "Группа политик". Группа с именем "New_policy" не будет возвращена.

- Такой же Поиск можно выполнять и в списках членства в группах.
- Теперь можно сортировать список групп по имени, используя стрелки справа от заголовка столбца имя, чтобы отсортировать список в порядке возрастания или убывания.

## <a name="group-member-search-and-filtering"></a>Поиск и фильтрация членов группы

### <a name="search-group-member-and-owner-lists"></a>Списки членов и владельцев группы поиска

Теперь вы можете искать членов определенной группы по имени и выполнять тот же поиск в списке владельцев группы. В новом интерфейсе, если ввести строку в поле поиска, будет автоматически выполнен поиск по функции StartsWith. Например, Поиск "Скотт" возвратит Скотт Вилкинсон.

![новые поиски подстрок в списках участников и владельцев группы](./media/groups-members-owners-search/members-list.png)

### <a name="filter-member-and-owners-list"></a>Фильтровать список элементов и владельцев

В дополнение к поиску теперь можно фильтровать списки членов и владельцев по типу пользователя. Это сведения, которые находятся в столбце Тип пользователя списка. Поэтому можно отфильтровать список по элементам и гостям, чтобы определить, есть ли в группе Гости.

### <a name="view-and-manage-membership"></a>Просмотр членства и управление им

Помимо просмотра непосредственных членов определенной группы, теперь можно просматривать список всех членов группы на странице члены. Список участников включает в себя все уникальные члены группы, включая все транзитивные члены.

Вы также можете выполнять поиск и фильтрацию списка "прямые участники" и список всех членов по отдельности. Фильтрация списка все члены не влияет на фильтры, применяемые в списке прямые члены.

## <a name="improved-group-member-counts"></a>Улучшенные счетчики членов группы

Мы улучшили страницу **Обзор** групп, чтобы обеспечить подсчет членов группы для групп всех размеров. Количество элементов можно увидеть даже для групп с более чем 1 000 элементами. Теперь можно увидеть общее число непосредственных членов группы и общее число членств (все уникальные члены группы, включая транзитивные элементы) на странице **Обзор** .

![Более высокая точность в количестве членств в группах](./media/groups-members-owners-search/member-numbers.png)
