# <a name="bulk-restore-deleted-users-in-azure-active-directory"></a>Массовое восстановление удаленных пользователей в Azure Active Directory

Azure Active Directory (Azure AD) поддерживает операции массового восстановления пользователей, а также скачивание списков пользователей, групп и участников групп.

## <a name="understand-the-csv-template"></a>Сведения о шаблоне CSV

Скачайте и заполните шаблон CSV, чтобы успешно восстанавливать пользователей Azure AD в пакетном режиме. Скачиваемый шаблон CSV может выглядеть так же, как в следующем примере:

![Электронная таблица для отправки со сносками, которые поясняют смысл каждой строки и столбца и значений в них](./media/users-bulk-restore/understand-template.png)

### <a name="csv-template-structure"></a>Структура шаблона CSV

В скачанном шаблоне CSV содержатся следующие строки.

- **Номер версии**: Первая строка с номером версии должна сохраняться в отправляемой версии CSV-файла.
- **Заголовки столбцов**. Формат заголовков столбцов: &lt;*имя элемента*&gt; [ИмяСвойства] &lt;*Required или пусто*&gt;. Например, `Object ID [objectId] Required`. Некоторые старые версии шаблона могут иметь небольшие отличия.
- **Строка примеров**. Мы включили в шаблон строку с примерами допустимых значений для каждого столбца. Строку примеров необходимо удалить, заменив реальными значениями.

### <a name="additional-guidance"></a>Дополнительные рекомендации

- Первые две строки шаблона отправки нельзя удалять или изменять, иначе отправка не будет обработана.
- Обязательные столбцы перечислены первыми.
- Мы не рекомендуем добавлять в шаблон новые столбцы. Любые дополнительные столбцы игнорируются и не обрабатываются.
- Мы рекомендуем скачивать свежие версии шаблона CSV как можно чаще.

## <a name="to-bulk-restore-users"></a>Массовое восстановление пользователей

1. [Войдите в свою организацию Azure AD](https://aad.portal.azure.com) с учетной записью администратора пользователей в организации Azure AD.
1. В Azure AD выберите **Пользователи** > **Удаленные**.
1. На странице **Удаленные пользователи** выберите **Групповое восстановление**, чтобы передать допустимый CSV-файл свойств восстанавливаемых пользователей.

    ![Выбор команды "Групповое восстановление" на странице "Удаленные пользователи"](./media/users-bulk-restore/bulk-restore.png)

1. Откройте шаблон CSV и добавьте строку для каждого пользователя, которого нужно восстановить. Единственное требуемое значение — **ObjectID**. Затем сохраните файл.

    :::image type="content" source="./media/users-bulk-restore/upload-button.png" alt-text="Выберите локальный CSV-файл, в котором перечислены пользователи, которых вы хотите добавить.":::

1. На странице **Групповое восстановление** в разделе **Отправить CSV-файл** найдите нужный файл. Когда вы выберете файл и нажмете **Отправить**, запустится проверка CSV-файла.
1. После проверки содержимого файла вы увидите сообщение **Файл успешно передан**. Если будут обнаружены ошибки, их нужно исправить для отправки задания.
1. Когда файл пройдет проверку, щелкните **Отправить**, чтобы запустить массовую операцию Azure для восстановления пользователей.
1. По завершении операции восстановления отобразится уведомление об успешном выполнении массовой операции.

При наличии ошибок можно загрузить и просмотреть файл результатов на странице **Результаты массовой операции**. Файл содержит причину каждой ошибки.

## <a name="check-status"></a>Проверка состояния

Состояние всех ожидающих выполнения массовых запросов можно просмотреть на странице **Результаты массовой операции**.

[![Проверьте состояние на странице результатов с массовыми операциями.](./media/users-bulk-restore/bulk-center.png)](./media/users-bulk-restore/bulk-center.png#lightbox)

Убедитесь в том, что восстановленные пользователи существуют в организации Azure AD, с помощью портала Azure или PowerShell.

## <a name="view-restored-users-in-the-azure-portal"></a>Просмотр восстановленных пользователей на портале Azure

1. [Войдите в центр администрирования Azure](https://aad.portal.azure.com) с учетной записью администратора пользователей в организации.
1. В области навигации выберите **Azure Active Directory**.
1. В разделе **Управление** выберите **Пользователи**.
1. В разделе **Показать** выберите **Все пользователи** и убедитесь в том, что восстановленные пользователи отображаются.

### <a name="view-users-with-powershell"></a>Просмотр пользователей с помощью PowerShell

Выполните следующую команду:

``` PowerShell
Get-AzureADUser -Filter "UserType eq 'Member'"
```

Вы увидите, что восстановленные пользователи указаны.
