# <a name="securing-external-collaboration-in-azure-active-directory-and-microsoft-365"></a>Обеспечение безопасности внешней совместной работы в Azure Active Directory и Microsoft 365

Безопасная совместная работа с внешними партнерами гарантирует, что у правых внешних партнеров будет соответствующий доступ к внутренним ресурсам в течение продолжительного периода времени. Благодаря глобальному подходу к управлению можно снизить риски безопасности, обеспечить соответствие требованиям и убедиться, что у вас есть доступ.

Нерегулирующая совместная работа ведет к недостаточной ясности в владении доступом и возможности предоставления конфиденциальных ресурсов. Переход в защищенную и контролируемую совместную работу может обеспечить наличие четких строк владения и учета для доступа внешних пользователей. В том числе:

* Управление внешними организациями и пользователями в них, которые имеют доступ к ресурсам.

* Проверка правильности доступа, проверки и временных ограничений, если это уместно.

* Предоставление бизнес-владельцам возможности управления совместной работой в пределах созданных защитных шин.

Если необходимо обеспечить соответствие платформам соответствия требованиям, то управление совместной работой позволяет подтверждать доступ к соответствующим требованиям.

Корпорация Майкрософт предлагает исчерпывающие наборы средств для безопасного внешнего доступа.  Служба совместной работы Azure Active Directory (Azure AD) B2B находится в центре любого внешнего плана совместной работы. Azure AD B2B можно интегрировать с другими инструментами в Azure AD, а также средствами служб Microsoft 365 Services для защиты внешнего доступа и управления им.

Этот набор документов предназначен для перехода от нерегламентированного или слабо специализированного внешнего взаимодействия к более безопасному состоянию.
