# <a name="add-or-update-a-users-profile-information-using-azure-active-directory"></a>Добавление или изменение данных профиля пользователя с помощью Azure Active Directory
Добавьте данные в профиль пользователя, в том числе изображение профиля, сведения о работе и некоторые параметры, с помощью Azure Active Directory (Azure AD). Дополнительные сведения о добавлении новых пользователей см. в статье [Добавление или удаление пользователей в Azure Active Directory](add-users-azure-active-directory.md).

## <a name="add-or-change-profile-information"></a>Добавление или изменение данных профиля
В профиле пользователя есть множество других данных, помимо тех, которые можно добавить во время создания пользователя. Все эти дополнительные сведения являются необязательными и добавляются по мере необходимости.

## <a name="to-add-or-change-profile-information"></a>Добавление или изменение данных профиля
1. Войдите в [портал Azure](https://portal.azure.com/) в качестве администратора пользователя для Организации.

2. Выберите **Azure Active Directory**, затем **Пользователи** и выберите пользователя. Например, _Alain Charon_.

    Появится страница **Alain Charon — профиль**.

    ![Страница профиля пользователя с доступными для изменения данными](media/active-directory-users-profile-azure-portal/user-profile-all-blade.png)

3. Щелкните **Изменить**, если необходимо добавить или изменить сведения в каждом из доступных разделов.

    ![Страница профиля пользователя с доступными для изменения разделами](media/active-directory-users-profile-azure-portal/user-profile-edit.png)

    - **Изображение профиля.** Выберите эскиз для учетной записи пользователя. Это изображение отображается в Azure Active Directory и на личных страницах пользователя, например на странице myapps.microsoft.com.

    - **Идентифицирует.** Добавление или обновление дополнительного значения идентификатора пользователя, например, фамилии в браке. Это имя можно задать независимо от значений имени и фамилии. Например, можно использовать его для включения инициалов, названия компании или для изменения отображаемой последовательности имен. В другом примере для двух пользователей, чьи имена имеют значение "Крис Green", можно использовать строку идентификатора, чтобы задать их имена "Крис B. Green" "Крис R. Green (Contoso)".

    - **Сведения о работе.** Добавьте сведения, связанные с работой, например должность, отдел или руководителя пользователя.

    - **Параметры.** Решите, может ли пользователь входить в клиент Azure Active Directory. Можно также указать географическое местонахождение пользователя.

    - **Контактные данные.** Добавьте соответствующие контактные данные для пользователя, за исключением некоторых телефонных или мобильных контактных данных пользователя (только глобальный администратор может обновить пользователей в ролях администратора).

    - **Контактная информация для проверки подлинности.** Проверьте актуальность номера телефона и адреса электронной почты пользователя. Azure Active Directory использует эти сведения для установления подлинности личности пользователя во время входа. Контактную информацию для проверки подлинности может изменять только глобальный администратор.

4. Щелкните **Сохранить**.

    Все изменения для пользователя сохранятся.

    >[!Note]
    >Используйте Windows Server Active Directory для обновления удостоверения, контактных данных или сведений о должности для пользователей, чьим главным источником является Windows Server Active Directory. После завершения обновления изменения будут отображаться после следующего цикла синхронизации.
