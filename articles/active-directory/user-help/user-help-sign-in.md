# <a name="sign-in-using-two-step-verification-or-security-info"></a>Вход с помощью двухфакторной проверки подлинности или сведений для защиты

После настройки двухфакторной проверки подлинности или сведений для защиты вы сможете войти в учетную запись, используя указанный метод проверки.

> [!Note]
> Если вы используете процедуру двухфакторной проверки подлинности, необходимо настроить методы проверки, следуя указаниям в статье [Настройка учетной записи для двухфакторной проверки подлинности](multi-factor-authentication-end-user-first-time.md).
>
> Если администратор включил функцию использования сведений для защиты, необходимо настроить методы проверки, выполнив пошаговые инструкции из этих статей:<ul><li>[Настройка использования приложения проверки подлинности в сведениях для защиты (предварительная версия)](security-info-setup-auth-app.md)</li><li>[Настройка использования обмена текстовыми сообщениями в сведениях для защиты (предварительная версия)](security-info-setup-text-msg.md)</li><li>[Настройка использования телефонного звонка в сведениях для защиты (предварительная версия)](security-info-setup-phone-number.md)</li><li>[Настройка использования ключа безопасности в сведениях для защиты](security-info-setup-security-key.md)</li></ul>

## <a name="sign-in-using-an-authenticator-app-notification-on-your-mobile-device"></a>Вход с помощью уведомления от приложения проверки подлинности на мобильном устройстве

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. В уведомлении, которое отправилось на мобильное устройство, выберите **Утвердить**.

## <a name="sign-in-using-an-authenticator-app-code-on-your-mobile-device"></a>Вход с помощью кода в приложении проверки подлинности на мобильном устройстве

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Откройте приложение проверки подлинности и введите случайным образом сформированный код для своей учетной записи в поле **Введите код**.

## <a name="sign-in-using-your-phone-number"></a>Вход с использованием номера телефона

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Ответьте на вызов, а затем следуйте инструкциям.

## <a name="sign-in-using-a-text-message"></a>Вход с использованием текстового сообщения

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Откройте сообщение и введите код из текста в поле **Введите код**.

## <a name="sign-in-using-a-security-key-at-the-lock-screen"></a>Вход с помощью ключа безопасности на экране блокировки

1. После регистрации ключа безопасности выберите изображение ключа безопасности на экране блокировки Windows 10.

2. Вставьте ключ безопасности в USB-порт устройства и войдите в Windows с помощью ПИН-кода ключа безопасности.

    ![Вход с помощью ключа безопасности на экране блокировки Windows 10](./media/security-info/security-info-windows-10-lock-screen-security-key.png)

## <a name="sign-in-using-a-security-key-and-the-microsoft-edge-browser"></a>Вход с помощью ключа безопасности и браузера Microsoft Edge

1. После регистрации ключа безопасности откройте браузер Microsoft Edge.

2. При появлении запроса на вход вставьте ключ безопасности в USB-порт устройства и войдите в Windows, используя ПИН-код ключа безопасности.

    ![Вход с помощью ключа безопасности в браузере Microsoft Edge](./media/security-info/security-info-edge-security-key.png)

    >[!NOTE]
    >Сведения о входе в систему с помощью приложения Microsoft Authenticator см. в статье [Вход в учетные записи с помощью приложения Microsoft Authenticator](user-help-auth-app-sign-in.md).

## <a name="sign-in-using-another-verification-method"></a>Выполнение входа с использованием другого способа проверки

Если по какой-либо причине вы не можете использовать основной метод входа, можно использовать другой ранее настроенный метод проверки.

1. Войдите в учетную запись, как обычно, и на странице **Двухшаговая проверка** щелкните ссылку на **изменение способа входа**.

    ![Изменение метода проверки при входе](media/security-info/two-factor-auth-signin-another-way.png)

    >[!Note]
    >Если вы не видите ссылку на **изменение способа входа**, значит, вы не настроили других методов проверки и для входа в свою учетную запись вам нужно обратиться за помощью к администратору. После того как администратор поможет вам выполнить вход, обязательно добавьте дополнительные методы проверки. Дополнительные сведения о добавлении методов проверки см. в статье [Управление параметрами двухфакторной проверки подлинности](multi-factor-authentication-end-user-manage-settings.md).
    >
    >Если вы видите ссылку на **изменение способа входа**, но не видите других методов проверки, тогда для входа в свою учетную запись нужно обратиться за помощью к администратору.

2. Выберите альтернативный метод проверки подлинности и продолжайте процесс двухфакторной проверки подлинности.

3. При необходимости можно обновить свои методы проверки, как только вы вернетесь в свою учетную запись. Дополнительные сведения о добавлении или изменении методов см. в статье [Управление параметрами двухфакторной проверки подлинности](multi-factor-authentication-end-user-manage-settings.md).
