# <a name="ui-framework-capabilities"></a>Возможности платформы пользовательского интерфейса

[!INCLUDE [Private Preview Notice](../../includes/private-preview-include.md)]

Инфраструктура пользовательского интерфейса в Службах коммуникации Azure позволяет создавать возможности коммуникации с помощью набора многократно используемых компонентов. Такие компоненты бывают двух видов: **Базовые** компоненты — это основные стандартные блоки пользовательского интерфейса, а их сочетания называются **составными** компонентами.

## <a name="ui-framework-composite-components"></a>Составные компоненты инфраструктуры пользовательского интерфейса

| Составной               | Описание                                               | Интернет   | Android | iOS   |
|-------------------------|-----------------------------------------------------------|-------|---------|-------|
| Составной компонент группового вызова | Упрощенный интерфейс исходящих вызовов с поддержкой голосовой связи и видеосвязи для вызовов Служб коммуникации Azure, использующий ресурсы разработки пользовательского интерфейса Fluent. Поддерживает групповые вызовы с использованием идентификатора группы Служб коммуникации Azure. Составной компонент поддерживает индивидуальные вызовы путем использования ссылки на удостоверение Служб коммуникации Azure или номер телефона для ТСОП (с помощью номера телефона, полученного через Azure).                                    | React |  |  |
| Составной компонент группового чата    | Упрощенный интерфейс чата для Служб коммуникации Azure, использующий ресурсы разработки пользовательского интерфейса Fluent. Он предоставляет простой клиент чата, который может подключаться к потокам Служб коммуникации Azure. Этот компонент позволяет пользователям отправлять сообщения и просматривать полученные сообщения с индикаторами ввода и уведомлениями о прочтении. Он масштабируется от индивидуальных вызовов до сценариев группового чата. Поддерживает один поток чата.                         | React |  |  |

## <a name="ui-framework-base-components"></a>Базовые компоненты инфраструктуры пользовательского интерфейса

| Компонент             | Описание                                                                                                                                                                                                                                                                        | Интернет   | Android | iOS |
|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------|---------|-----|
| Поставщик вызовов    | Основной инициализирующий компонент для вызовов. Обязательный компонент, который затем инициализирует поверх себя другие компоненты. Обрабатывает базовую логику для инициализации вызывающего клиента с помощью маркеров доступа Служб коммуникации Azure. Поддерживает присоединение группы. | React | Н/Д     | Н/Д |
| Элементы управления мультимедиа   | Позволяют пользователям управлять текущим вызовом путем переключения звука, включения или выключения видео и завершения вызова.                                                                                                                                                              | React | Н/Д     | Н/Д |
| Коллекция мультимедиа   | Отображение всех участников вызова в одной коллекции. Коллекция поддерживает отображение участников как в статическом формате, так и в видеоформате. Для участников с поддержкой видео отображается видео.                                                                                                                | React | Н/Д     | Н/Д |
| Параметры микрофона | Выберите микрофон, который будет использоваться для вызова. Этот элемент управления можно использовать до и во время вызова для выбора устройства микрофона.                                                                                                                                               | React | Н/Д     | Н/Д |
| Параметры камеры     | Выберите камеру, которая будет использоваться для видеовызова. Этот элемент управления можно использовать до и во время вызова для выбора устройства видео.                                                                                                                                             | React | Н/Д     | Н/Д |
| Параметры устройств     | Объединяет параметры микрофона и камеры в одном компоненте.                                                                                                 | React | Н/Д     | Н/Д |
| Поставщик чата       | Основной инициализирующий компонент для чата. Обязательный компонент, который затем инициализирует поверх себя другие компоненты. Обрабатывает базовую логику для инициализации клиента чата с помощью маркера доступа Служб коммуникации Azure и потока, к которому он будет присоединен.                                     | React | Н/Д     | Н/Д |
| Поле отправки          | Компонент для ввода, который позволяет пользователям отправлять сообщения в поток чата. В качестве входных данных поддерживаются текст, гиперссылки, эмодзи и другие символы Юникода, включая другие алфавиты.                                                                                                                         | React | Н/Д     | Н/Д |
| Поток чата           | Компонент потока, который отображает пользователю полученные и отправленные сообщения с данными об отправителе. Поток поддерживает индикаторы ввода и уведомления о прочтении. Вы можете прокручивать эти потоки для просмотра журнала чата.
| Список участников      | Отображает всех участников вызовов или потока чата в виде списка.  | React | Н/Д     | Н/Д |
