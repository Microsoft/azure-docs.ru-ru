# <a name="supported-browsers"></a>Поддерживаемые браузеры 

Azure зрения поддерживает следующие браузеры. Мы советуем использовать самые последние браузеры, совместимые с вашей с операционной системой. 

* Microsoft Edge (последняя версия);
* Safari (последняя версия, только для Mac);
* Chrome (последняя версия);
* Firefox (последняя версия).

## <a name="chrome-incognito-mode"></a>Режим режиме инкогнито Chrome

 Для работы зрения Studio необходимо отключить от Chrome режиме инкогнито блокировки файлов cookie сторонних производителей.

:::image type="content" source="./media/supported-browsers/incognito-chrome.png" alt-text="Снимок экрана с изображением Chrome.":::

## <a name="chromium-edge-inprivate-mode"></a>Chromium пограничной режим InPrivate

Для работы зрения Studio необходимо отключить Chromium ребра с помощью функции запрета отслеживания.
