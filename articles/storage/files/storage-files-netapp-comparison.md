# <a name="azure-files-and-azure-netapp-files-comparison"></a>Сравнение файлов и Azure NetApp Files Azure

В этой статье содержится сравнение файлов и Azure NetApp Files Azure. 

Большинство рабочих нагрузок, требующих облачного хранилища файлов, хорошо работают на файлах или Azure NetApp Files Azure. Чтобы определить, что лучше подходит для вашей рабочей нагрузки, ознакомьтесь со сведениями, приведенными в этой статье. Дополнительные сведения см. в документации по [службе файлов](https://docs.microsoft.com/azure/storage/files/) и [Azure NetApp Files](https://docs.microsoft.com/azure/azure-netapp-files/) Azure.
