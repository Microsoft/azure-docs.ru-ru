Эта страница представляет собой индекс встроенных определений политик в [Политике Azure](../../governance/policy/overview.md) для службы хранилища Azure. Дополнительные встроенные компоненты Политики Azure для других служб см. в статье [Встроенные определения Политики Azure](../../governance/policy/samples/built-in-policies.md).
