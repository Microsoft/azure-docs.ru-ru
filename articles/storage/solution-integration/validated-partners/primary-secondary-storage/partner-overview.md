# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Партнеры службы хранилища Azure для основного и дополнительного хранилища

В этой статье рассматриваются партнерские компании Майкрософт, которые доставляют решения для NAS или SAN в локальной среде, в Azure или гибридном решении, которое использует службу хранилища Azure в качестве экономичного уровня. Эти решения позволяют клиентам использовать одно и то же решение в любой из своих сред.
