# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Партнеры службы хранилища Azure для основного и дополнительного хранилища

В этой статье описываются партнерские компании Майкрософт, которые обеспечивают подключение к сети (NAS) или сети хранения данных (SAN). Решение может быть в локальной среде, в Azure или в гибридном решении, которое использует службу хранилища Azure в качестве экономичного уровня. Эти решения позволяют клиентам использовать одно и то же решение в любой из своих сред.
