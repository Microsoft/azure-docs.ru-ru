# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Партнеры по архивации, резервному копированию и аварийному восстановлению службы хранилища Azure

В этой статье описываются партнеры Майкрософт, интегрированные с хранилищем Azure для архивных и BCDR рабочих нагрузок. Они используют преимущества масштабирования и [экономичности](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) службы хранилища Azure для эффективного решения проблем с резервным копированием, создания сайта аварийного восстановления или архивирования неиспользуемого содержимого для долгосрочного хранения. Благодаря всем стандартам соответствия требованиям, предъявляемым к хранилищу Azure, а также к [неизменяемому хранилищу](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) и управлению жизненным циклом, эти решения могут легко заменить резервные копии на ленте и предложить экономичный сайт восстановления по запросу.
