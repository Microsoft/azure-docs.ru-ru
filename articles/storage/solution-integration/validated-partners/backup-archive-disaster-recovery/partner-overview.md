# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Партнеры по архивации, резервному копированию и аварийному восстановлению службы хранилища Azure

В этой статье описываются партнеры Майкрософт, интегрированные с хранилищем Azure для архивации, архивации и для рабочих нагрузок непрерывности бизнес-процессов и аварийного восстановления (BCDR). Эти решения партнеров используют преимущества масштабирования и экономичности службы хранилища Azure. Решения можно использовать для решения задач резервного копирования, создания сайта аварийного восстановления или архивации неиспользуемого содержимого для долгосрочного хранения. Благодаря всем стандартам соответствия требованиям хранилища Azure и функциям Azure, таким как [неизменяемое хранилище](../../../blobs/storage-blob-immutable-storage.md) и [Управление жизненным циклом](../../../blobs/storage-lifecycle-management-concepts.md), эти решения могут легко заменять резервные копии на ленте и предлагать экономичный сайт восстановления по запросу.
