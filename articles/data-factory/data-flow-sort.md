# <a name="sort-transformation-in-mapping-data-flow"></a>Преобразование «Сортировка» в потоке данных сопоставления

[!INCLUDE[appliesto-adf-asa-md](includes/appliesto-adf-asa-md.md)]

Преобразование «Сортировка» позволяет сортировать входящие строки в текущем потоке данных. Можно выбрать отдельные столбцы и отсортировать их в порядке возрастания или убывания.

> [!NOTE]
> Сопоставление потоков данных выполняется в кластерах Spark, которые распределяют данные между несколькими узлами и секциями. Если вы решили повторно секционировать данные в последующем преобразовании, вы можете потерять сортировку из-за перегруппировка данных.

## <a name="configuration"></a>Конфигурация

![Параметры сортировки](media/data-flow/sort.png "Сортировка")

**Без учета регистра:** Следует ли игнорировать регистр при сортировке строк или текстовых полей

**Сортировать только в секциях:** Так как потоки данных выполняются в Spark, каждый поток данных делится на разделы. Этот параметр сортирует данные только в входящих секциях, а не при сортировке всего потока данных. 

**Условия сортировки:** Выберите столбцы, по которым выполняется сортировка, и порядок сортировки. Порядок определяет приоритет сортировки. Выберите, будут ли отображаться значения NULL в начале или в конце потока данных.

### <a name="computed-columns"></a>Вычисляемые столбцы

Чтобы изменить или извлечь значение столбца перед применением сортировки, наведите указатель мыши на столбец и выберите "вычисляемый столбец". Откроется построитель выражений, чтобы создать выражение для операции сортировки вместо использования значения столбца.

## <a name="data-flow-script"></a>Скрипт потока данных

### <a name="syntax"></a>Синтаксис

```
<incomingStream>
    sort(
        desc(<sortColumn1>, { true | false }),
        asc(<sortColumn2>, { true | false }),
        ...
    ) ~> <sortTransformationName<>
```

### <a name="example"></a>Пример

![Параметры сортировки](media/data-flow/sort.png "Сортировка")

Скрипт потока данных для указанной выше конфигурации сортировки приведен в следующем фрагменте кода.

```
BasketballStats sort(desc(PTS, true),
    asc(Age, true)) ~> Sort1
```
