# <a name="azure-automanage-for-virtual-machines-best-practices---windows-server"></a>Рекомендации по управлению Azure для виртуальных машин — Windows Server

Эти службы Azure автоматически подключаются при использовании автоматического управления для виртуальных машин на виртуальной машине Windows Server. Они необходимы для нашего технического документа, который можно найти в нашей [облачной инфраструктуре внедрения](/azure/cloud-adoption-framework/manage/azure-server-management).

Для всех этих служб мы будем автопереходить на автоматическую настройку, отслеживать отклонения и устранять их при обнаружении отклонения. Дополнительные сведения об этом процессе см. в разделе [Управление Azure для виртуальных машин](automanage-virtual-machines.md).

## <a name="supported-windows-server-versions"></a>Поддерживаемые версии Windows Server

Управление поддерживает следующие версии Windows Server:

- Windows Server 2012 и R2
- Windows Server 2016
- Windows Server 2019
- Выпуск Windows Server 2019 Azure
