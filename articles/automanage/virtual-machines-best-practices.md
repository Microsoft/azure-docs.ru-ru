# <a name="azure-automanage-for-virtual-machines-best-practices"></a>Рекомендации по управлению Azure для виртуальных машин


Эти службы Azure автоматически подключаются при использовании автоматического управления для виртуальных машин. Они необходимы для нашего технического документа, который можно найти в нашей [облачной инфраструктуре внедрения](/azure/cloud-adoption-framework/manage/azure-server-management).

Для всех этих служб мы будем осуществлять автоматическое подключение, автоматическую настройку, отслеживать отклонения и исправлять, если будет обнаружено отклонение. Дополнительные сведения об этом процессе см. в разделе [Управление Azure для виртуальных машин](automanage-virtual-machines.md).


## <a name="participating-services"></a>Участвующие службы

|-----------|---------------|----------------------|-------------------------|
|Мониторинг VM Insights    |Azure Monitor для виртуальных машин отслеживает производительность и работоспособность виртуальных машин, включая выполняющиеся процессы и зависимости от других ресурсов. Дополнительные [сведения.](../azure-monitor/vm/vminsights-overview.md)    |Рекомендации для виртуальных машин Azure — Рабочая среда    |нет    |

Дополнительные [сведения.](../backup/backup-azure-vms-introduction.md) Расходы основываются на количестве и размере защищаемых виртуальных машин. Дополнительные [сведения.](https://azure.microsoft.com/pricing/details/backup/)    |Рекомендации для виртуальных машин Azure — Рабочая среда    |Да    |
|Центр безопасности Azure    |Центр безопасности Azure — это единая система управления безопасностью инфраструктуры, которая повышает уровень безопасности центров обработки данных и обеспечивает расширенную защиту от угроз для гибридных рабочих нагрузок в облаке. Дополнительные [сведения.](../security-center/security-center-introduction.md)  Служба управления настраивает подписку, в которой находится ваша виртуальная машина, на бесплатное предложение центра безопасности Azure. |Microsoft Antimalware    |Антивредоносное ПО Майкрософт для Azure предоставляет бесплатную защиту в реальном времени, которая помогает обнаруживать и устранять вирусы, шпионское ПО и другие вредоносные программы. Оно создает предупреждения о попытках установки или запуска известных вредоносных или нежелательных программ в системе Azure. |Управление обновлениями    |Управление обновлениями в службе автоматизации Azure можно использовать для управления обновлениями операционной системы для виртуальных машин. Благодаря ей вы сможете быстро оценить состояние доступных обновлений на всех компьютерах агентов и управлять установкой необходимых обновлений на серверах. |Отслеживание изменений & инвентаризации    |Отслеживание изменений и реестр сочетают функции отслеживания изменений и инвентаризации, позволяя отслеживать изменения в виртуальной машине и инфраструктуре серверов.
