В этой статье приводятся общие сведения о различных типах ошибок и рекомендации по обработке распространенных ошибок входа.

## <a name="msal-error-handling-basics"></a>Основные сведения об обработке ошибок MSAL

Исключения в библиотеке проверки подлинности Майкрософт (MSAL) предназначены для устранения неполадок разработчиками приложений, а не для отображения конечным пользователям. Сообщения об исключениях не локализованы.

При обработке ошибок и исключений вы можете различать их по типам исключений и кодам ошибки.  Список кодов ошибок см. в разделе [коды ошибок аутентификация Azure AD и авторизации](../articles/active-directory/develop/reference-aadsts-error-codes.md).

Во время входа в систему могут происходить ошибки, связанные с согласованностью, условным доступом (MFA, управлением устройствами, ограничениями на основе расположения), выдачей и возвратом токенов и свойствами пользователей.
