Параметры DNS не являются обязательной частью этой конфигурации, но при необходимости разрешения имен между виртуальными машинами требуется DNS. Если указать значение, DNS-сервер не создается. Необходимо указать IP-адрес DNS-сервера, который может разрешать имена для ресурсов, к которым вы подключаетесь.

После создания виртуальной сети можно добавить IP-адрес DNS-сервера для обработки разрешения имен. Откройте параметры виртуальной сети, выберите DNS-серверы и добавьте IP-адрес DNS-сервера, который требуется использовать для разрешения имен.

1. На портале щелкните виртуальную сеть.
2. На странице виртуальной сети в разделе **Параметры** выберите **DNS-серверы**.
3. Добавьте DNS-сервер.
4. Чтобы сохранить параметры, нажмите кнопку **сохранить** в верхней части страницы.
