Пакет SDK для пространственных привязок Azure для версии 2,9 или более поздней должен быть настроен в [Unity 2020,3 (LTS)](https://unity3d.com/unity/whats-new/2020.3.0) с помощью [подключаемого модуля Unity XR](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) , использующего следующие пакеты:

- AR Foundation: [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@4.0/manual/index.html)
- Подключаемый модуль XR для Windows (для поддержки устройств HoloLens): [4.4.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr@4.4/manual/index.html)
- Подключаемый модуль Аркоре XR (для поддержки устройств Android): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arcore@4.0/manual/index.html)
- Подключаемый модуль ARKit XR (для поддержки устройств iOS): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arkit@4.0/manual/index.html)

Пакет SDK для пространственных привязок Azure версии 2,8 или более ранней не поддерживает [инфраструктуру подключаемых модулей Unity XR](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) и должен быть настроен в [Unity 2019,4 (LTS)](https://unity.com/releases/2019-lts) с помощью следующих пакетов:

- AR Foundation: [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html)
- Windows Mixed Reality (для поддержки устройств HoloLens): [4.2.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html)
- Подключаемый модуль Аркоре XR (для поддержки устройств Android): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html)
- Подключаемый модуль ARKit XR (для поддержки устройств iOS): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html)
