Пакет SDK для пространственных привязок Azure должен быть настроен в <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019,4 (LTS)</a> с помощью следующих пакетов:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- Подключаемый модуль Аркоре XR (для поддержки устройств Android): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- Подключаемый модуль ARKit XR (для поддержки устройств iOS): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> В настоящее время пакет ASA SDK не поддерживает <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">инфраструктуру подключаемых модулей Unity XR</a>
