- Чтобы получить доступ к службе Data Box Edge, ваша организация должна иметь подписку Соглашение Enterprise (EA) или поставщика облачных решений (CSP). Дополнительные сведения см. в статье [Регистрация для получения подписки Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Так как эта служба управления размещается в Azure, она защищена функциями безопасности Azure. Дополнительные сведения о функциях безопасности, предоставляемых Azure, см. в [Microsoft Azure центре управления безопасностью](https://azure.microsoft.com/support/trust-center/security/).
- Для операций управления пакетом SDK можно получить ключ шифрования для ресурса в **свойствах устройства**. Ключ шифрования можно просмотреть только в том случае, если у вас есть разрешения на API Graph ресурсов.
