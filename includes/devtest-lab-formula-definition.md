Формула в Azure DevTest Labs — это список значений свойств, используемых по умолчанию при создании виртуальной машины. При создании виртуальной машины на основе формулы значения по умолчанию можно использовать "как есть" или изменить. Формулы, так же как [пользовательские образы](../articles/devtest-labs/devtest-lab-create-template.md) и [образы Marketplace](../articles/devtest-labs/devtest-lab-configure-marketplace-images.md), обеспечивают быструю подготовку виртуальных машин. Формулы, так же как и [пользовательские образы](../articles/devtest-labs/devtest-lab-create-template.md), позволяют создать базовый образ из VHD-файла. Затем базовый образ можно использовать для подготовки новой виртуальной машины. Чтобы выбрать подходящий вариант для своей среды, ознакомьтесь со статьей [Сравнение пользовательских образов и формул в DevTest Labs](../articles/devtest-labs/devtest-lab-comparing-vm-base-image-types.md).
