В таблице ниже перечислены некоторые основные различия между Студией машинного обучения (классической) и службой "Машинное обучение Azure".

| Компонент | Студии машинного обучения (классической) | Машинное обучение Azure |
|---| --- | --- |
| Интерфейс перетаскивания | Классический интерфейс | Обновленный интерфейс — [конструктор Машинного обучения Azure](../articles/machine-learning/concept-designer.md)| 
| Пакеты SDK для кода | Не поддерживается | Полная интеграция с пакетами SDK [Машинного обучения Azure для Python](/python/api/overview/azure/ml/) и [R](https://github.com/Azure/azureml-sdk-for-r) |
| Эксперимент | Масштабируемый (ограничение на объем данных для обучения: 10 ГБ) | Масштабирование с помощью целевого объекта вычислений |
| Обучение целевых объектов вычислений | Собственный целевой объект, поддерживающий только ЦП | Обширный набор настраиваемых [целевых объектов вычислений для обучения](../articles/machine-learning/concept-compute-target.md#train). Включает поддержку GPU и ЦП. | 
| Развертывание целевых объектов вычислений | Собственный формат веб-службы, без возможности настройки | Обширный набор настраиваемых [целевых объектов вычислений для развертывания](../articles/machine-learning/concept-compute-target.md#deploy). Включает поддержку GPU и ЦП. |
| ML-конвейер | Не поддерживается | Создание гибких, модульных [конвейеров](../articles/machine-learning/concept-ml-pipelines.md) для автоматизации рабочих процессов |
| MLOps | Развертывание базовой модели и управление ею; развертывания с использованием только ЦП | Управление версиями сущностей (модель, данные, рабочие процессы), автоматизация рабочих процессов, интеграция с инструментами непрерывной интеграции и поставки, развертывания с использованием ЦП и GPU, [а также многие другие возможности](../articles/machine-learning/concept-model-management-and-deployment.md) |
| Формат модели | Собственный формат, только Студия (классическая) | Несколько поддерживаемых форматов в зависимости от типа задания обучения |
| Автоматизированное обучение моделей и настройка гиперпараметров |  Не поддерживается | [поддерживается](../articles/machine-learning/concept-automated-ml.md). Возможно использовать подход без написания кода или Code First. | 
| Обнаружение смещения данных | Не поддерживается | [Поддерживается](../articles/machine-learning/how-to-monitor-datasets.md) |
| Проект маркировки данных | Не поддерживается | [Поддерживается](../articles/machine-learning/how-to-create-labeling-projects.md) |
| Управление доступа на основе ролей | Только роль участника и владельца | [Гибкое определение роли и управление RBAC](../articles/machine-learning/how-to-assign-roles.md) |
| Коллекция решений ИИ | Поддерживается ([https://gallery.azure.ai/](https://gallery.azure.ai/)) | Не поддерживается <br><br> Обучайтесь с помощью [примеров записных книжек SDK для Python](https://github.com/Azure/MachineLearningNotebooks). |
