- Уровень производительности диска можно понизить только один раз в 12 часов.

## <a name="change-performance-tier-without-downtime-preview"></a>Изменение уровня производительности без простоя (Предварительная версия)

Как правило, для изменения уровня производительности потребуется отменить выделение виртуальной машины или отсоединить диск. Но если включить эту функцию предварительного просмотра, вам не нужно будет освобождать виртуальную машину или отсоединить диск, чтобы изменить уровень. Вы можете зарегистрироваться для просмотра [здесь](https://aka.ms/liveperftiersignup).

Предварительный просмотр имеет следующие ограничения.
- Доступно только в EastUS2EUAP регионе.
- Сейчас недоступно для общих дисков
- `2020-12-01`Для изменения уровней производительности без простоя необходимо использовать шаблоны Azure Resource Manager с API.
