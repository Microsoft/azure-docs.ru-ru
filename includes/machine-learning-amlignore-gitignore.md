Чтобы предотвратить включение ненужных файлов в моментальный снимок, создайте файл игнорирования (`.gitignore` или `.amlignore`) в каталоге. Добавьте исключаемые файлы и каталоги в этот файл. Дополнительные сведения о синтаксисе, который будет использоваться в этом файле, см. в разделе [синтаксис и шаблоны](https://git-scm.com/docs/gitignore) для `.gitignore`. Файл `.amlignore` использует тот же синтаксис. _Если оба файла существуют, используется файл `.amlignore`, а файл `.gitignore` не используется._
