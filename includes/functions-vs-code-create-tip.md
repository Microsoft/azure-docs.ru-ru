> [!TIP]
> По умолчанию ресурсы Azure, необходимые для приложения функции, создаются на основе указанного имени приложения функции. По умолчанию они также создаются в той же новой группе ресурсов, что и приложение-функция. Если необходимо настроить имена этих ресурсов либо повторно использовать существующие ресурсы, [опубликуйте проект с дополнительными параметрами создания](../articles/azure-functions/functions-develop-vs-code.md#enable-publishing-with-advanced-create-options).
