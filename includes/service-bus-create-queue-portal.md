## <a name="create-a-queue-in-the-azure-portal"></a>Создание очереди на портале Azure
1. На странице **Пространство имен служебной шины** в меню навигации слева выберите **Очереди**.
1. На странице **Очереди** на панели инструментов выберите **+ Очередь**.
1. Введите **имя** очереди, остальные значения по умолчанию не изменяйте.
1. Выберите **Создать**.
 
    ![Очереди -> + Очередь -> Введите имя -> Создать](./media/service-bus-create-queue-portal/create-queue3.png)
