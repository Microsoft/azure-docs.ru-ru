## <a name="set-up-your-device"></a>Настройка устройства

Чтобы развернуть приложения на устройстве HoloLens, необходимо связать его с компьютером.

1. На устройстве HoloLens перейдите в раздел **Параметры -> Обновление и безопасность -> Для разработчиков**.
2. Нажмите **Связать** и не закрывайте этот экран, пока не введете PIN-код в Visual Studio во время первого развертывания.
