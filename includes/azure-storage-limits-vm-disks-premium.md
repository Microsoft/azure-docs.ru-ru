| Ресурс | Ограничение |
| --- | --- |
| Общая емкость диска на одну учетную запись |35 ТБ |
| Общая емкость моментального снимка на одну учетную запись |10 ТБ |

<sup>1</sup> Под термином *входящие данные* подразумеваются все данные из запросов, отправляемые в учетную запись хранения. Под термином *входящие данные* подразумеваются все данные из запросов, получаемые из учетной записи хранения.
