Ниже указаны ограничения, применяемые к Azure Site Recovery.

| Идентификатор ограничения | Ограничение |
| --- | --- |
| Количество хранилищ на одну подписку |500 |     
| Количество серверов на одно хранилище Служб восстановления |250 |
| Количество групп защиты на одно хранилище Служб восстановления |Без ограничений |
| Количество планов восстановления на одно хранилище Служб восстановления |Без ограничений |
| Количество серверов на одну группу защиты |Без ограничений |
| Количество серверов на один план восстановления |100 |
