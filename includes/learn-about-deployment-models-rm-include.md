> [!NOTE]
> В Azure предлагаются две модели развертывания для создания ресурсов и работы с ними: [развертывание с помощью Azure Resource Manager и классическая модель развертывания](../articles/azure-resource-manager/management/deployment-models.md). В этой статье описывается модель развертывания с помощью Resource Manager. Мы рекомендуем использовать для новых развертываний модель развертывания с помощью Resource Manager вместо классической модели развертывания.
