Подключитесь к виртуальной машине Windows с помощью протокол удаленного рабочего стола (RDP) через IP-адрес, переданный во время создания виртуальной машины.

1. На клиенте Откройте RDP. 
1. Перейдите в раздел **Пуск** и введите **mstsc**.
1. В области **Подключение к удаленному рабочему столу** введите IP-адрес виртуальной машины и учетные данные доступа, которые использовались в файле параметров шаблона виртуальной машины, а затем нажмите кнопку **подключить**.

   ![Снимок экрана подключение к удаленному рабочему столу панели для подключения по протоколу RDP к виртуальной машине Windows.](media/azure-stack-edge-gateway-connect-vm-windows/connect-vm-rdp-1.png)

   > [!NOTE]
   > Может потребоваться утвердить подключение к ненадежному компьютеру. 

Теперь вы вошли в виртуальную машину, которая работает на устройстве.
