Производительность управляемого диска Azure задается при создании диска в виде его уровня производительности. Уровень производительности определяет операции ввода-вывода и пропускную способность для управляемого диска. При установке подготовленного размера диска автоматически выбирается уровень производительности. Уровень производительности можно изменить при развертывании или впоследствии без изменения размера диска. Дополнительные сведения о уровнях производительности см. в статье [уровни производительности для управляемых дисков](../articles/virtual-machines/disks-change-performance.md).
