Настройте правила оповещений, чтобы получать сообщения о ситуациях, связанных с потреблением ресурсов на вашем устройстве. Вы можете настроить правила оповещений для отслеживания устройства на возникновение ситуаций, которые приводят к отправке оповещений. Подробные сведения об оповещениях см. в статье [Создание и просмотр оповещений метрик, а также управление ими с помощью Azure Monitor](../articles/azure-monitor/alerts/alerts-metric.md).
