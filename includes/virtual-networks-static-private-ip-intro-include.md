Виртуальной машине автоматически назначается частный IP-адрес из указанного диапазона в зависимости от подсети, в которой развернута виртуальная машина. Виртуальная машина оставляет адрес до тех пор, пока виртуальная машина не будет удалена. Azure динамически назначает следующий доступный частный IP-адрес из подсети, в которой создается виртуальная машина. Если вам нужен конкретный IP-адрес из подсети, назначенной виртуальной машине, назначьте статический IP-адрес.
