> [!NOTE]  
> Часовой пояс по умолчанию, используемый в выражениях CRON, — время в формате UTC. Если нужно использовать другой часовой пояс в выражении CRON, создайте для приложения-функции параметр приложения с именем WEBSITE_TIME_ZONE. См. сведения о [часовых поясах NCRONTAB](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
